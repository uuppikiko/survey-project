# Survey Data Collection App

A web-based survey application that collects user responses and saves them to a CSV file.

## About This Project

This project was built to demonstrate a full-stack data collection workflow — the same kind of system used in professional market research. Users fill out a survey form in their browser, and the responses are automatically saved to a CSV file via a Python backend.

## Technologies Used

- **HTML** — survey form structure
- **CSS** — page styling and layout
- **JavaScript** — form submission and communication with the backend
- **Python (Flask)** — backend server that receives and saves responses
- **CSV** — data storage format for collected responses

## How It Works

1. User opens `index.html` in their browser
2. User fills out the survey form and clicks Submit
3. JavaScript sends the response to the Python backend
4. Flask saves the response as a new row in `responses.csv`
